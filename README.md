# _Pizza Order_

#### _Allow inputs to order pizzas, 4/12/19_

#### By _**Justin Kerntz**_

## Description

_This project allows users to add multiple pizzas to an order that shows all customizable options picked and the price or each item. I also included a total amount due._

## Setup/Installation Requirements

* _Clone Repo from GitHub_
* _Open index.html in Chrome browser_

_Other browsers may experience errors_

## Known Bugs

_No way to go back to add more items after submit, also no delete feature so choice wisely._
